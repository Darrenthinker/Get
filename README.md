## 🚀 使用方法

1. **访问网站**: 双击 `index.html` 在浏览器中打开
2. **浏览分类**: 使用左侧边栏导航浏览不同分类
3. **搜索内容**: 使用顶部搜索框搜索相关知识
4. **使用工具**: 点击工具卡片使用实用功能
5. **查看详情**: 点击知识卡片查看详细内容

## 💡 特色说明

这个知识库的设计完全参考了你提供的两个网站(AMZ123和WL123)的多层级导航结构，实现了：

1. **深度分类体系**: 每个货代板块下都有5-6个详细的子分类
2. **专业内容丰富**: 每个子分类包含多篇专业文章
3. **工具集成**: 类似WL123的工具导向设计
4. **现代化界面**: 比原网站更现代的UI设计
5. **移动端友好**: 完全响应式的设计

这样的架构让货代新手可以从入门到精通，系统性地学习各个领域的专业知识，同时也方便有经验的从业者快速查找特定信息。
